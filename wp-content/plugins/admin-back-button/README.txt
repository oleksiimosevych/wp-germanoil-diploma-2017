== Description ==

A simple back button added to all post types edit page. You can filter and navigate to posts lists without worrying about updating the post and lose the posts before.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `back-button.zip` to the `/wp-content/plugins/` directory and extract
1. Activate the plugin through the 'Plugins' menu in WordPress
